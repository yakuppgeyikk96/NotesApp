# NotesApp Repo

This repo was created for having practice redux-toolkit through a basic notes app.

## Installation

Clone the project

```bash
git clone ""
```

## Usage

Open the folder in VSCode (or different editor you prefer)

For Linux:
```linux
cd NotesApp
code .
```
Run:
```
npm install
```

After the installation of packages run:
```
npm run dev
```
